# Chez Dark Shade Ark

This is a repository for the Adventuron source code for *Chez Dark Shade Ark*, which was submitted to [EctoComp 2025]. The deadline was October 30, 2025, and the initial release was October 31, 2025. It placed 16th of 32 entries. It is currently on version 3.0.0, with few (if any) future releases planned.

*Chez Dark Shade Arc is an 11-room game with 13 points. It is in the mold of [Us Too](https://github.com/andrewschultz/us-too), [Why Pout](https://github.com/andrewschultz/why-pout) and [Go-Strange-Ghost Range](https://github.com/andrewschultz/go-strange-ghost-range).

A runnable web version is [here](https://andrewschultz.itch.io/chez-dark-shade-arc), with a downloadable HTML file, and with individual files more easily downloadable.

## Game Details

*Chez Dark Shade Arc* is a wordplay game which requires you to find two-word phrases. It has 11 core points and no bonus points.

## Walkthroughs and maps

The repository contains source code, a game map, a logic map and a walkthrough, all of which should be on the Itch page, too. The graphics subdirectory has, well, the graphics.

The walkthrough.txt file will guide you through how to get all the points. There are no bonus points.

Version 1 was written late August 2025, then the second half of September 2025, then published October 30, 2025 for EctoComp Petite Mort, where it placed 16th of 32 entries.
Version 2 was published after EctoComp finished, on December 3, 2025.
Version 3 was published January 16, 2026. Its main feature is to track half-right guesses, but it also has general improvements thanks to testers' feedback.
